Existem 2 tipos de Sass:

style.sass

style.scss


Dicas Sass:

  • style.sass:

    • Para fazer o processamento do ficheiro style.sass, temos que escrever no terminal da nossa pasta que contem o ficheiro sass: sass --watch style.sass:style/style.css

      • Isto vai fazer com que os comandos que escrevemos no ficheiro sass (style.sass), sejam processados num ficheiro css à parte!
      • Neste caso, estámos a criar uma pasta (com os "dois pontos" : ) e a transferir os comandos do sass para o ficheiro css dentro da pasta style ( :style/style.css )
    • Linkar o sass:
      • Para linkar o sass, temos que escrever o nosso link, e no href vamos à pasta style, ao ficheiro css (como se fosse a linkar o css normal) rel="stylesheet" href="./style/style.css">
    • Não há " {} "
      • Ver o botão "Ir para: Sass" para perceber melhor.
    • Não há " ; "
      • Ver o botão "Ir para: Sass" para perceber melhor.
    • Extensão do VScode recomendada para o Sass:
      • "Sass"
    • Sair do processamento sass no terminal:
      • CTRL+C dentro do terminal
