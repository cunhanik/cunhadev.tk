Existem 2 tipos de Sass:

style.sass

style.scss


Dicas Sass:

  • style.sass:

    • Para fazer o processamento do ficheiro style.sass, temos que escrever no terminal da nossa pasta que contem o ficheiro sass: sass --watch style.sass:style/style.css

      • Isto vai fazer com que os comandos que escrevemos no ficheiro sass (style.sass), sejam processados num ficheiro css à parte!
      • Neste caso, estámos a criar uma pasta (com os "dois pontos" : ) e a transferir os comandos do sass para o ficheiro css dentro da pasta style ( :style/style.css )
    • Linkar o sass:
      • Para linkar o sass, temos que escrever o nosso link, e no href vamos à pasta style, ao ficheiro css (como se fosse a linkar o css normal) rel="stylesheet" href="./style/style.css">
    • Não há " {} "
      • Ver o botão "Ir para: Sass" para perceber melhor.
    • Não há " ; "
      • Ver o botão "Ir para: Sass" para perceber melhor.
    • Extensão do VScode recomendada para o Sass:
      • "Sass"
    • Sair do processamento sass no terminal:
      • CTRL+C dentro do terminal

  • style.scss:

    • Para fazer o processamento do ficheiro style.sass, temos que escrever no terminal da nossa pasta que contem o ficheiro scss: sass --watch style.scss:css/style.css

      • Isto vai fazer com que os comandos que escrevemos no ficheiro sass (style.sass), sejam processados num ficheiro css à parte!
      • Neste caso, estámos a criar uma pasta (com os "dois pontos" : ) e a transferir os comandos do sass para o ficheiro css dentro da pasta css ( :css/style.css )
    • Linkar o sass:
      • Para linkar o sass, temos que escrever o nosso link, e no href vamos à pasta style, ao ficheiro css (como se fosse a linkar o css normal) rel="stylesheet" href="./style/style.css">
    • Não há " {} "
      • Ver o botão "Ir para: Sass" para perceber melhor.
    • Não há " ; "
      • Ver o botão "Ir para: Sass" para perceber melhor.
    • Extensão do VScode recomendada para o Sass:
      • "Sass"
    • Sair do processamento sass no terminal:
      • CTRL+C dentro do terminal

Dicas Sass GERAIS:

  • Variaveis:
    • Utilizamos variaveis para facilitar o trabalho, vamos usar como exemplo o ficheiro: sass-variaveis.html
      • Para escrever uma variavel no scss, usamos o $: $btn-color1: #ff4043; $btn-color2: #00ff00; $btn-width: 150px;
      • Depois quando quisermos usar essa cor e essa width, fazemos: .btn1 { width: $btn-width; background-color: $btn-color1; }
    • O Sass mostra os erros no terminal

      • Carreguem no botão "Ir para: SASS VARIAVEIS"

  • Nesting:
    • Com o Nesting, o nosso css fica mais organizado! Escrevemos o nome da classe e fazemos tudo dentro dela

      • Ver Ex Nesting

  • Import:
    • O import funciona da seguinte maneira:
      • Criamos o ficheiro principal: sass-import.scss
      • Criamos um ficheiro scss de cores: sass-import-colors.scss
      • Criamos um ficheiro de uma pagina que vamos estilizar (ex: pagina Home): sass-import-home.scss
        • Agora, no ficheiro "sass-import-colors.scss" criamos as variaveis das cores que mais vamos usar no nosso projeto:

          $primary-color: red;

          $secondary-color: orange;

      • No ficheiro "sass-import-home.scss" criamos o estilo do nosso exemplo:

        .text-click {

        color: $secondary-color;

        font-size: 22px;

        }

        .btn {

        padding: 15px;

        background-color: black;

        color: $secondary-color;

        border: 1px solid $primary-color;

        &:hover {

        background-color: white;

        color: $primary-color;

        border-color: black;

        }

        }

      • No ficheiro principal "sass-import.scss" importamos tudo o que usamos
        • //import colors

          @import "./sass-import-colors.scss";

        • //import home

          @import "./sass-import-home.scss";

        Feito isto, é so fazer o processamento normal:

        Na consola da pasta onde está inserido o nosso ficheiro:

        sass --watch sass-import.scss:sass-import.css

      Ver Ex import

  • mixin:
    • Serve para facilitar e resumir o codigo.
      • Vamos criar um ficheiro de exemplo, usar: variaveis, nesting, import e mixin
      • Criamos o ficheiro: ex_mixin.html e ex_mixin_home.scss
        • No ficheiro "ex_mixin.html":
          • Criamos a nossa pagina html e add:
            • 2 divs com classes diferentes, 1 delas com um h1 e um p; A outra com um botão
              • div class="txt" div
                • h1 class="title" Se quiseres comprar h1
                • p class="paragraph" Clica no botão p
              • div class="btn" div
                • button class="btn-style" Click-Here button
        • No ficheiro "ex_mixin_home.scss":

            • Fazemos o mixin, que vai ajudar-nos a poupar linhas de código.

          • @mixin text($font-size, $font-family, $color) {

            font-size: $font-size;

            font-family: $font-family;

            color: $color;

            }

            @mixin btn($font-size, $color, $background-color, $border, $cursor, $padding-right) {

            font-size: $font-size;

            color: $color;

            background-color: $background-color;

            border: $border;

            cursor: $cursor;

            padding-right: $padding-right;

            }

            .txt {

            .title {

            @include text(24px, Arial, $primary-color);

            }

            .paragraph {

            @include text(18px, Arial, $secondary-color);

            }

            }

            .btn {

            .btn-style {

            @include btn(16px, white, black, 1px solid $primary-color, pointer, 30px);

            &:hover {

            @include btn(16px, black, $secondary-color, 2px solid darkgrey, pointer, 10px);

            }

            }

            }

      • Criamos o ficheiro: ex_mixin_variaveis.scss e ex_mixin.scss

        • No ficheiro "ex_mixin_variaveis.scss", escrevemos as nossas variaveis:

        • $primary-color: red;
        • $secondary-color: blue;

        No ficheiro "ex_mixin.scss", escrevemos os nossos imports:

        • @import "./ex_mixin_variaveis.scss";
        • @import "./ex_mixin.scss";

      Feito isto, agora é so processar o ficheiro principal (QUE CONTEM OS IMPORTS):

      No terminal da pasta onde contem o nosso ficheiro:

      sass --watch ex_mixin.scss:ex_mixin.css

      linkar o css da pagina html: link rel="stylesheet" href="./ex_mixin.css" />

      Ver Ex mixin

  • Herança e Extends
    • Vamos criar 1 ficheiro html e scss:
      • No ficheiro html: ex_heranca-extends.html:
        • Criamos 3 butoes:
        • o 1º e 3º com class btn e class da propria cor
        • o 2º só com class da cor

          • button class="btn btn-red">I'M RED button

          button class="btn-green">I'M GREEN button

          button class="btn btn-blue">I'M BLUE button

        • Processamos o sass

          • No terminal fazemos: sass --watch ex_heranca-extends.scss:ex_heranca-extends.css

        • Linkamos o css

          • No html linkamos: link rel="stylesheet" href="./ex_heranca-extends.css" />

      • No ficheiro sass: ex_heranca-extends.scss:

        • Parte da HERANCA:

            • Primeiro vamos estilizar a classe btn:

          • .btn {

            padding: 15px 10px;

            border-radius: 25px;

            width: 125px;

            display: block;

            margin-bottom: 10px;

            cursor: pointer;

            • Segundo, dentro dessa classe, vamos estilizar a class de cada botão individual com essa classe btn:

            Para fazermos isso, precisamos de usar o &.nome-da-propria-class

          • &.btn-red {

            background-color: red;

            color: white;

            }

            &.btn-blue {

            background-color: blue;

            color: white;

            }

            }

        • Parte do EXTENDS:

          • Terceiro, e fora da classe, colocamos o nosso 2º botão:

          • .btn-green {

            @extend .btn;

            background-color: green;

            color: white;

            }

      Carregar no botão "ex_heranca-extends"
