    Background-color:

    No nosso ficheiro de css, escrevemos:

    body{background-color: gray;} para deixarmos a pagina toda com o fundo cinzento

    h1{background-color: #000000;} para deixarmos o elemento titulo com o fundo preto

    Background-image

    No nosso ficheiro de css, escrevemos:

    body{

    background-color:gray;

    background-repeat: no-repeat; #Para o nosso fundo não se repetir

    background-size: cover; #Tamanho do background cover(capa)

    color: white; #Para as letras ficarem brancas

    }

    Seletores filhos div > li{}

    div>li{} Este seletor serve para estilizarmos todos os elementos li dentro das divs.

    Especificidade

    O CSS vai sempre colocar o estilo mais específico. Regra da pontuação (numero da esquerda é o maior, numero da direita é o menor):

    - Cada elemento/pseudoelemento -> d = +1 (0, 0, 0, 1); Exemplo: h1 { background: blue; }

    Cada classe/pseudoclasse/atributo -> c = +1 (0, 0, 1, 0); Exemplo: .red { background: red; }

    - Cada id -> b = +1 (0, 1, 0, 0); Exemplo: id="titulo" / declaração id: #titulo{ background: lightblue; }

    - Estilo inline(no proprio HTML) -> a = +1 (1, 0, 0, 0); Exemplo: style="background: green;"

    Box Model

    A estrutura de um Box Model é:

    CONTENT

    Vamos dar uma largura (width) de 5% para o nosso h5, assim o elemento só ocupa aquele espaço

    Vamos adicionar uma margin de 50px auto 20px auto; (Significa que em cima temos 50px, direita auto, baixo 20px, esquerda auto)

    Vamos agora adicionar uma borda ao nosso h5: border 2px dashed red

    Agora vamos usar o padding que é o espaço que fica entre o h5 e a borda: 10px(cima) 15px(direita) 10px(baixo) 15px(esquerda);
