Exercicio flexbox+grid

Dicas flexbox:

  • display: flex; || Torna o nosso container flex, e com isso podemos fazer muitas coisas:

    • flex-direction:
      • flex-direction: column || Coloca os nossos Items em coluna
      • flex-direction: row || Coloca os nossos Items em linha
      • flex-direction: column-reverse || Coloca os nossos items em coluna mas no sentido contrário, de baixo para cima
      • flex-direction: row-reverse || Coloca os nossos items em linha mas no sentido contrário, da direita para a esquerda
    • gap:
      • row-gap: 15px || Acrescenta um espaço em linha entre os items que estiverem em coluna (flex-direction: column \ flex-direction: column-reverse)
      • column-gap: 15px || Acrescenta um espaço em coluna entre os items que estiverem em linha (flex-direction: row \ flex-direction: row-reverse)
      • gap: 0px 15px || Acrescenta um espaço de 0px em linha para os items- (flex-direction: column) e 15px em coluna para os items- (flex-direction: row)
      • gap: 20px || Acrescenta um espaço de 20px em linha para os items (flex-direction: column) e 20px em coluna para os items (flex-direction: row)
    • Justify-content:
      • Justify-content: center || Vai centrar os items horizontalmente por default. Se mudarmos o flex-direction: column, ele centra os items verticalmente
      • Justify-content: flex-end \ end || Vai colocar os items no fim da pagina (horizontalmente)
      • Justify-content: flex-start \ start || Vai colocar os items no inicio da pagina (horizontalmente)
      • Justify-content: space-around || Vai colocar os items com o mesmo espaço na esquerda e na direita, consequentemente cria um espaço entre eles
      • Justify-content: space-between || Vai colocar os items com o maximo de espaço entre eles, consequentemente vão ficar encostados nas laterais
      • Justify-content: space-evenly || Vai colocar os items com o mesmo espaço entre eles, consequentemente cria um espaço (igual) nas laterias
    • Align-items:
      • Align-items: center || Vai colocar os items ao centro verticalmente. É preciso ter o min-height: 100vh;
      • Align-items: flex-start || Vai colocar os items no inicio verticalmente. É preciso ter o min-height: 100vh;
      • Align-items: flex-end || Vai colocar os items no fim verticalmente. É preciso ter o min-height: 100vh;
      • Align-items: stretch || Vai colocar os items esticados verticalmente. É preciso ter o min-height: 100vh;
    • flex-wrap:
      • flex-wrap: nowrap || Em termos de responsividade, o nowrap faz com que os items à medida que estámos a diminuir a tela, diminuam tambem
      • flex-wrap: wrap || Em termos de responsividade, o wrap faz com que os items à medida que estámos a diminuir a tela, passem para a linha de baixo
      • flex-wrap: wrap-reverse || Em termos de responsividade, o wrap faz com que os items à medida que estámos a diminuir a tela, passem para a linha de cima
    • flex-basics:
      • flex-basics: 200px || Basicamente é a largura do item (width) para usarmos isto, podemos não usar width ou comentar. Tal como fizemos no Exercicio 3
    • flex-grow:
      • flex-grow: 0 || fica tudo normal (valor default)
      • flex-grow: 1 || ocupa todo o espaço disponivel na horizontal
    • flex-shrink:
      • flex-shrink: 0 || O item não vai diminuir ao contrario dos outros, conforme formos diminuindo a resolução. Para usarmos isto, podemos não usar flex-wrap: wrap ou comentar. Tal como fizemos no Exercicio 3
      • flex-shrink: 1 || O item vai diminuir igual aos outros, conforme formos diminuindo a resolução. Para usarmos isto, podemos não usar flex-wrap: wrap ou comentar. Tal como fizemos no Exercicio 3
      • flex-shrink: 2 || O item diminui mais do que os outros, conforme formos diminuindo a resolução. Para usarmos isto, podemos não usar flex-wrap: wrap ou comentar. Tal como fizemos no Exercicio 3
    • flex:
      • flex: 1 0 200px || o 1 é o flex-grow, ou seja: Ocupa todo o espaço disponivel. | o 0 é o flex-shrink, ou seja: O item não vai diminuir. | o 200px é o flex-basics, ou seja: O item vai ocupar 200px largura.

        Para vermos melhor:

        No Exercicio 3: tiramos o flex-wrap, usamos o flex-direction: column no container e mudamos a height: 100px dos items.

        Com o flex-basics ou o 3º numero do flex (neste caso 200px), tambem podemos ver que o item 2 fica responsivo verticalmente. Se baixarmos o tamanho na vertical, vemos que ele mantem os 200px

    • order:
      • order: 1 || O order 1 coloca o item na 1ª posição
      • order: 2 || O order 2 coloca o item na 2ª posição
      • order: 3 || O order 3 coloca o item na 3ª posição
    • Align-items:

        • Para testar isto, No Exercicio 3, no .container vamos: Tirar o flex-direction: column ou alterar para row

        Para testar isto, No Exercicio 3 vamos: alterar a width do .item para 20%

      • Align-self: flex-start || Alinha verticalmente o item para o topo
      • Align-self: flex-end || Alinha verticalmente o item para o bottom
      • Align-self: center || Alinha verticalmente o item para o centro
    • align-content:
      • align-content: center; || Alinha verticalmente o item como o align-items, mas reduz o espaço que fica no meio se utilizarmos gap
      • align-content: flex-start; || Alinha verticalmente o item para o topo como o align-items, mas reduz o espaço que fica no meio se utilizarmos gap
      • align-content: flex-end; || Alinha verticalmente o item para o bottom como o align-items, mas reduz o espaço que fica no meio se utilizarmos gap
