Dicas Bootstrap:

  • Instalação do Bootstrap:

    Podemos fazer de duas maneiras:

    1. Instalação via link:
      • Instalação CSS:

        No head da pagina html: < link href="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-1BmE4kWBq78iYhFldvKuhfTAU6auU8tT94WrHftjDbrCEXSU1oBoqyl2QvZ6jIW3" crossorigin="anonymous"/>


      • Instalação JavaScript:

        Antes de fechar o body, no fim da pagina html: script src="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/js/bootstrap.bundle.min.js" integrity="sha384-ka7Sk0Gln4gmtz2MlQnikT1wXgYsOg+OMhuP+IlRH9sENBO0LRn5q+8nbTov4+1p" crossorigin="anonymous"> < /script>


    2. Instalação via npm:
      • No terminal: npm install bootstrap
  • Bootstrap Grid:
    • Breakpoints || É o que ajuda na responsividade, quebra o item em certos px

        • Podemos escrever 6 Breakpoints:

      • class="col-xs coll"
        • A Responsividade do xs (extra small) é aos 0px
      • class="col-sm coll"
        • A Responsividade do sm (small) é aos 576px ;
      • class="col-md coll"
        • A Responsividade do md (medium) é aos 768px;
      • class="col-lg coll"
        • A Responsividade do lg (large) é aos 992px;
      • class="col-xl coll"
        • A Responsividade do xl (extra large) é aos 1200px;
      • class="col-xxl coll"
        • A Responsividade do xxl (extra extra large) é aos 1400px;
    • gutter || É parecido com o gap, mas o gutter dá um espaço entre o paragrafo e a outra coluna.
      • É colocado dentro da class: class="row gx-1">

        Podemos escrever 3 tipos de gutter:

      • g-3 || gutter geral (laterais, cima e baixo); Normalmente o valor default é 3
      • gx-1 || gutter em linha (horizontal)
      • gy-2 || gutter em coluna (vertical)

  • Imagens responsivas e utilidades:
    • Imagens:
      • class="img-fluid" || Usamos esta classe para a imagem ficar responsiva, automaticamente com o "fluid" a imagem tambem ocupa 100% de largura
      • class="rounded float-start" || Alinhar as imagens no inicio da tela
      • class="rounded float-end" || Alinhar as imagens no fim da tela
      • class="rounded mx-auto d-block" || Alinhar as imagens no centro da tela
    • utilidades:
      • margin-> m-|| Aqui as medidas são todas em rem;
        • my-3 || Margem na vertical(cima e baixo) de 3rem
        • mx-2 || Margem na horizontal (laterais) de 2 rem
        • m-5 || Margem geral de 5rem: 5rem é o valor maximo
        • mt-1 || Margem top de 1rem
        • mb-4 || Margem bottom de 4rem
        • ms-3 || Margem start (left) de 3rem
        • me-2 || Margem end (direita) de 2rem
      • padding
        • py-2 || Padding na vertical(cima e baixo) de 2rem
        • px-3 || Padding na horizontal(laterais) de 3rem
        • p-5 || Padding geral de 5rem: 5rem é o valor maximo
        • pb-4 || Padding bottom de 4rem
        • ps-3 || Padding start (left) de 3rem
        • pe-2 || Padding end (direita) de 2rem
      • display
        • d-flex || Display flex (elementos lado a lado)
        • d-block || Display block (elementos em coluna
        • d-none || Display none, faz com que o elemento desapareça

  • BootStrap+Sass
    • Para usarmos o bootstrap com o Sass, não podemos instalar o bootstrap com link. Tem que ser via npm:
      • Abrir a consola, na pasta correta e: npm install bootstrap
    • Feita a instalação, foram criadas varias pastas e ficheiros. Agora temos que linkar o bootstrap:

        • Como sempre, no head da pagina:

        • Ordem de ligação:
          1. ./node_modules/
          2. bootstrap/
          3. dist/
          4. css/
          5. bootstrap.css
      • link rel="stylesheet" href="./node_modules/bootstrap/dist/css/bootstrap.css"

    • Agora linkar o JavaScript:

        • Como sempre, antes de fechar o /body:

        • Ordem de ligação:
          1. ./node_modules/
          2. bootstrap/
          3. dist/
          4. css/
          5. bootstrap.css
      • script src="./node_modules/bootstrap/dist/js/bootstrap.js "> < /script>
    • Para manipular as coisas, tem que ser na pasta: node_modules/bootstrap/scss/_variables.scss
      • Procurar as variaveis:
        • $primary: $blue !default;
        • $secondary: $gray-600 !default;
        • $success: $green !default;
        • $info: $cyan !default;
        • $warning: $yellow !default;
        • $danger: $red !default;
        • $light: $gray-100 !default;
        • $dark: $gray-900 !default;
      • Ver a que cor pertence cada variavel:
      • E se quisermos mudar alguma cor, alteramos aqui:

        • O blue pode ser vermelho, verde, branco, etc.

        • $blue: #0d6efd !default;
        • $indigo: #6610f2 !default;
        • $purple: #6f42c1 !default;
        • $pink: #d63384 !default;
        • $red: #dc3545 !default;
        • $orange: #fd7e14 !default;
        • $yellow: #ffc107 !default;
        • $green: #198754 !default;
        • $teal: #20c997 !default;
        • $cyan: #0dcaf0 !default;
      • Para usarmos a cor que queremos, fazemos:
        • class="container bg-primary" || A class container, tem background (bg) primary-$blue-#0d6efd (cor que quisermos)
        • Para fazermos a alteração da cor da variavel, temos que importar o Sass:
          • importar o Sass: sass --watch ./node_modules/bootstrap/scss:./node_modules/bootstrap/dist/css
