Exercicio de responsividade

No css do exercicio:

  1. Diminuir as imagens para:
    • width: 765px; height: 400px;
  2. Configurar o texto:
    • font-size: 20px; font-family: Arial, Helvetica, sans-serif;
  3. Responsivo a 795px, que é quando a imagem fica quase colada à borda da tela
    • @media (max-width: 795px) {
    • width: 500px;
    • font-size: 16px;
  4. Responsivo a 510px, que é quando a imagem fica novamente quase colada à borda da tela
    • @media (max-width: 510px) {
    • width: 100%;
    • height: auto;
  5. Responsivo a 420px, texto centrado
    • @media (max-width:420){
    • text-align: center;

Dicas css:

  • min-width: 250px || é a largura minima o item, utilizámos isto no exercicio 5 Grid.
